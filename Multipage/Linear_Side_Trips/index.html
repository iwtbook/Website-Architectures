Linear with Side Trips - Multipage

This website is similar to the pure linear structure in that there is a straight path from the first page to the last page of the site, but along the way there are opportunities for users to take "side trips". This is often used on sites where the user is filling out a form where selecting certain options will require the user to fill out additional information, or on sites where something new / complex is introduced and the user has the option to read more information on that topic.
